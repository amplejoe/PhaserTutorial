ShoeBox is a free Adobe Air based app for Windows and Mac OSX with game and ui related tools.

Installation (page: http://renderhjs.net/shoebox/)
   
1.   Install the Adobe AIR runtime
2.   Uninstall previous versions before 3.6.5 (new certificate)
3.   Intall the ShoeBox.air file
